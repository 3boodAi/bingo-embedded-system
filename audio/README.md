# Audio Assets

DFPlayer Mini source files live in `MP3/`.

Copy the contents of `MP3/` to the MicroSD card's `/MP3` folder before inserting it into the DFPlayer Mini.

Expected production files:

- `0001.mp3` through `0075.mp3`: number announcements.
- `0101.mp3`, `0102.mp3`, `0103.mp3`: winner announcements for player slots 1-3.
