# Audio Assets

DFPlayer Mini source files live in `MP3/`.

Copy the contents of `MP3/` to the MicroSD card's `/MP3` folder before inserting it into the DFPlayer Mini.

Expected production files:

- `0000.mp3` through `0099.mp3`: clean number-only announcements.
- `0200.mp3`: "Bingo".
- `0201.mp3`, `0202.mp3`, `0203.mp3`: player 1, 2, and 3 winner announcements.
- `0204.mp3`: "Good game".
- `0205.mp3`: "Good luck".
- `0210.mp3`: full countdown audio, "5, 4, 3, 2, 1, go".
